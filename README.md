# Dauphin

Dauphin is an iOS app for Tamkang University students, providing easy access to course schedules, library services, campus events, and student ID barcode generation.

## Features

- **Course Schedule**: View and manage your class schedule with intuitive weekly and daily views
- **Library Services**: Access library resources and information
- **Campus Events**: Stay updated with university events and activities
- **Student ID Barcode**: Generate and display your student ID barcode for quick scanning
- **iOS Widgets**: Quick access to upcoming courses and student ID directly from your home screen
- **Offline Support**: Core features work without an internet connection
